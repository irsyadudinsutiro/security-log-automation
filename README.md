# Security Log Automation Pipeline

Automated Microsoft 365 security and audit log extraction and reporting pipeline.

## Tools & Technologies
- PowerShell
- Microsoft Graph REST API
- OAuth 2.0 (Client Credentials Flow)
- Microsoft 365 / Azure AD

## What it does
Authenticates via OAuth 2.0, extracts security and audit logs from Microsoft 365 
through Graph API, transforms the data, and schedules automated weekly governance 
report distribution — eliminating 70%+ of manual reporting effort.

## Note
Sensitive values (tenant ID, client ID, client secret) have been replaced with 
placeholders for security purposes.
